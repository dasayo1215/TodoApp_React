# TodoApp_React

Reactのコンポーネント構造や状態管理、イベント処理を学習し、関数コンポーネントでToDoアプリを再構築しました。
useState による状態管理と props によるデータ受け渡しで UI を部品化しています。

## 🔗 動作デモ＆ソースコード（StackBlitzで開く）
[https://stackblitz.com/edit/vitejs-vite-4guec2dm](https://stackblitz.com/edit/vitejs-vite-4guec2dm)

## 📘 学習ポイント
関数コンポーネントとpropsの理解
useStateによる状態管理
イベント処理とリスト描画の基礎

## ⚙️ 実装機能
新規タスク登録
タスクの完了／削除／復元
タスク数の制限・一意ID付与（crypto.randomUUID）
